How the Tree Hole Works

1) Write a letter anonymously. 2) Watch it glide into the Tree Hole. 3) Our counselor reads and responds through the admin dashboard. 4) When a reply is ready, it remains paired with your original letter for mindful continuity.

Why There Is a Daily Limit

To make sure every letter receives a thoughtful response, the administrator can gently limit how many letters arrive from the same IP each day. This keeps the pace humane for both the counselor and the seekers.
